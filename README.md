# Smart College Complaint and Resolution System


## Project Description
This project is developed to manage college complaints in an online way instead of manual registers.  
Students can submit complaints, and the admin can view and resolve them.


I selected this project to learn how real-time applications work using Flask and MySQL.


## Objectives
- To avoid manual complaint registers
- To make complaint tracking easy
- To provide transparency between students and college management


## Users
- Student
- Admin
  
  
## Features
- Student can register complaints
- Admin can view complaints
- Complaint status tracking
- Simple and user-friendly interface
  

## Technologies Used
- Frontend: HTML, CSS
- Backend: Python (Flask)
- Database: MySQL
  

## How to Run
1. Install Python
2. Install Flask


python -m pip install flask
3. Run the project  
python app.py


4. Open browser and go to
http://127.0.0.1:5000
